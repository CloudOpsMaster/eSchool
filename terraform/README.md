# Softserve2021_HW_2

Requirements.

1. Terraform v1.0.3
2. gsp.json file with valid GCP credentials

How to run.

1. Adjust instance, project and user in variables.tf
2. Adjust variables at #VARIABLES section in first_run.sh
3. terraform init
4. terraform apply

Application will start on 8080 tcp port
